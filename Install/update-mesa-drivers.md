Update/Install MESA Graphics Driver

```
sudo apt update
```

```
sudo apt upgrade
```

```
glxinfo | grep Mesa
```

```
sudo add-apt-repository ppa:kisak/kisak-mesa
```

```
sudo apt update
```

```
sudo apt install mesa
```

```
glxinfo | grep Mesa
```

 Now Mesa Drivers are updated/installed succesfully
